PPMcrypt
功能包含->加密->計算SHA256 Hash->檔名亂碼->分成5個檔案


dePPM
功能包含->讀入data資料夾內所有檔案->用Hash分辨檔案順序->解密->PPM復原

--------------------------------------------------------------------------
Warning:data資料夾內不可含有其他檔案，不然解密會錯誤
使用Release的.exe檔時，.pdb也要在同一個資料夾內，不然無法執行
